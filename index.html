Problem Statement

In a very promising and novel technology known as Electric/Ion Propulsion, efficiency (including ionizing and thrust efficiency) is the main factor inhibiting such technologies with big theoretical advantages from being useful and efficient when actually built. This affects aerospace/aeronautical companies, propulsion labs, and the future of transportation.

Our Solution

Our Python system calculates optimal thrust and cathode voltage for ion thrusters based on user inputs, providing the voltage profile, thrust, and efficiency. Using PyTorch and PINNs, it acts as a user-friendly tool for efficient propulsion system optimization. That is, We will use code to optimize input voltages along the cathode of a discharge channel for an Ion Thruster/Electric propulsion (EHD Thrusters) to maximize outflow efficiency. This is done by training and using Physics-informed Neural Networks (PINNs) in Python to find the best function of distance along the discharge tunnel to determine the voltage at each point that maximizes thrust. This would also generally mean that it ensures that the propellant gas is maximally ionized and that the ions are accelerated outward at the fastest velocity.

More about PINNs (Physics-Informed Neural Networks)

Physics-informed neural networks (PINNs) are neural networks that incorporate physical laws described by differential equations into their loss functions to guide the learning process toward solutions that are more consistent with the underlying physics. PINNs can be used to: Approximate solutions to partial differential equations (PDEs) and ordinary differential equations (ODEs). Solve inverse problems, such as estimating model parameters from limited data. PINNs are a class of physics-informed machine learning methods that seamlessly integrate physics knowledge with data. Often, PINNs get compared with purely data-driven methods and traditional numerical methods for solving problems involving PDEs and ODEs. Unlike purely data-driven approaches, which learn mathematical relationships solely from input and output data, PINNs:
  • Use prior physics knowledge.
  • Make more accurate predictions outside of the training data set.
  • Are more effective with limited or noisy training data.
Unlike traditional numerical methods for solving differential equations, such as finite element analysis for PDEs, PINNs:
  • Are mesh-free.
  • Can approximate high-dimensional PDE solutions.
  • Can solve for missing model parameters such as unknown PDE or ODE coefficients.
  • Can solve ill-posed problems where no boundary data exists.
  • Can easily incorporate sparse or noisy measurements.
